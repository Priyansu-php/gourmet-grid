# Gourmet Grid

A modern **online food ordering and delivery system** designed to make food discovery, ordering, and order management simple and efficient for users, restaurants, and administrators.

## Overview

Gourmet Grid is a database-driven web project built to simulate a real-world food delivery platform. The system allows customers to browse food items, place orders, manage their profiles, and track order-related activity. It also supports backend management for restaurants, menus, and order records.

This project demonstrates the practical use of:

* database design and management
* web-based application development
* CRUD operations
* user interaction workflows
* structured project organization

## Features

### User Side

* User registration and login
* Browse restaurants and food items
* View menu details and prices
* Add items to cart
* Place food orders
* Manage user profile
* View order history

### Admin / Management Side

* Manage food items and menus
* Manage restaurant-related records
* View and manage customer orders
* Update item availability
* Maintain database records efficiently

## How to Run the Project

### Prerequisites

* Install **XAMPP** / **WAMP** / **MAMP** or any local PHP server
* Install **MySQL**
* Install **Git**
* A modern web browser

### Setup Steps

1. Clone the repository:

   ```bash
   git clone git@github.com:Priyansu-php/gourmet-grid.git
   ```

2. Move the project folder into your local server directory:

   * For XAMPP: `htdocs`
   * For MAMP: `htdocs`

3. Start:

   * Apache
   * MySQL

4. Create a database in MySQL.

5. Import the SQL file into the database using phpMyAdmin or MySQL CLI.

6. Update the database connection file with your local database credentials.

7. Open the project in your browser:

   ```text
   http://localhost/gourmet-grid/
   ```
